# writing — 文章・日本語ライティング

AI が書いた日本語を人間が書いた文章に近づける横断的なライティング支援。主体の不在・命題型見出し・両論併記・記号アーティファクト等を直します。`references/` を同梱した自己完結スキルです。

## 導入

```
/plugin marketplace add gghatano/skill-repository
/plugin install writing@gghatano-skills
```

スキルは `/writing:stop-ai-slop-jp` で呼び出せます。下書き・編集・レビュー時に併用してください。`research` プラグインの `report-review` は、このスキルが導入されていれば自動で連携します。

## 含まれるスキル
