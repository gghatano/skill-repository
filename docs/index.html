導入手順

Claude Code の /plugin でマーケットプレイスを追加し、目的のプラグインをインストールします。 スキルはプラグイン名前空間付き（/<plugin>:<skill>）で呼び出せます。

1マーケットプレイスを追加

このリポジトリをプラグインの配布元として登録します（初回のみ）。

/plugin marketplace add gghatano/skill-repository

2プラグインをインストール

必要なプラグインだけ入れられます（下の「プラグイン」の各カードからコマンドをコピーできます）。

/plugin install <name>@<marketplace>

3スキルを使う

インストール済みプラグインのスキルは名前空間付きで呼び出します（例）。

/research:report-review
