generation-cycle

5 スキルを逐次委譲する合成データ生成の PM オーケストレーター。

/generation:generation-cycle
SKILL.md を見る ↗

概要

合成データ生成パイプラインの PM (project manager) エージェントとして起動する。必要に応じて原資料から input/ を作成し、5 つの SKILL (input-prepare → spec-ingest → generation-design → generate → evaluate-refine) を順次サブエージェントに委譲し、各ステップの Acceptance Criteria 充足を確認しながら end-to-end で実行する。

できること

  • input/ の起動前チェックを行う
  • 5 スキルをサブエージェントに逐次委譲する
  • 各ステップの Acceptance Criteria を確認する
  • quality_gate.json で改善ループを制御する
  • 進捗・完了報告をまとめる

使いどころ

タスクディレクトリを指定して input-prepare から evaluate-refine までを end-to-end で実行させたいとき、PM エージェントとして起動する。

入出力・連携

引数はタスクのルートディレクトリ。自身はコードを書かず、各ステップをサブエージェント（最大 5 体）に委譲し、成果物存在と quality_gate.json を確認しながら再生成は最大 2 回まで制御する。

このスキルを導入

プラグイン generation の一部です。

/plugin install generation@gghatano-skills
