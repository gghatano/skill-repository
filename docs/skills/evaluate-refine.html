evaluate-refine

合成データを仕様・制約に照らし評価し必要なら生成器を改善するスキル。

/generation:evaluate-refine
SKILL.md を見る ↗

概要

タスクディレクトリ ($ARGUMENTS) の合成データを、仕様・サンプル・制約に照らして評価し、必要に応じて generator.py を修正する。evaluation_report.md と constraints_check.csv を出力する。

できること

  • evaluate.py を実装し評価する
  • スキーマ・分布・相関を評価する
  • 制約違反を constraints_check.csv に出す
  • evaluation_report.md に所見をまとめる
  • quality_gate.json で改善要否を判定する

使いどころ

generate 完了後、成果物の品質と制約充足を評価し、軽微な問題は generator.py に最小修正するとき（step4）に使う。

入出力・連携

入力は input/sample_data・output のデータ・work 仕様・generator.py、出力は evaluate.py・evaluation_report.md・constraints_check.csv・quality_gate.json。最終的な再生成ループの判断は PM に委ねる。

このスキルを導入

プラグイン generation の一部です。

/plugin install generation@gghatano-skills
