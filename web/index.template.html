導入手順

このリポジトリを一度取得し、install_skills.py でバンドルまたは個別スキルを引き込み先リポジトリのルートへコピーします。 既存ファイルは --force を付けない限り上書きしません。

1このリポジトリを取得

引き込み元（マスター）を手元に用意します。以後 <skill-repository> はこのクローン先を指します。

git clone https://github.com/gghatano/skill-repository

2利用可能な一覧を確認

引き込めるバンドルと汎用スキルの一覧を表示します。

python3 <skill-repository>/scripts/install_skills.py --list

3引き込み先のルートで実行

バンドルごと引き込む場合（下の「バンドル」の各カードからコマンドをコピーできます）。

python3 <skill-repository>/scripts/install_skills.py --bundle <name> --into .

個別スキルだけ引き込む場合（--skill は繰り返し指定可。規約・エージェントも要るなら --with-companions）。

python3 <skill-repository>/scripts/install_skills.py --skill <name> --into .

バンドル

役割ベースに汎化した共通部分（スキル本体・規約・サブエージェント）を、目的単位でまとめて引き込むための定義です。

汎用スキル

特定プロジェクトのパスに依存しない、役割ベースの正本スキルです。単体でも、バンドルの一部としても引き込めます。
