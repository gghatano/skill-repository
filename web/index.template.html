導入手順

Claude Code の /plugin でマーケットプレイスを追加し、目的のプラグインをインストールします。スキルは名前空間付き /<plugin>:<skill> で呼び出せます。

  1. 1

    マーケットプレイスを追加

    配布元として登録します（初回のみ）。

    /plugin marketplace add gghatano/skill-repository
  2. 2

    プラグインを導入

    必要なものだけ選べます。各カードからコピー可。

    /plugin install <name>@<marketplace>
  3. 3

    スキルを使う

    名前空間付きで呼び出します（例）。

    /research:report-review
